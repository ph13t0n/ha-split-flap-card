# Split-Flap Card v0.1.0-beta.16

Beta 16 is a release-blocker validation build focused on editor usability, visual depth controls and split-flap display polish before a future stable `v0.1.0` release.

## Summary

This beta addresses findings from the Beta 15 Home Assistant validation round.

It is intended for testing through CDN or HACS beta/pre-release flow before promoting anything to a public stable release.

## Added

- Display frame control in the visual editor.
- Decorative screw control in the visual editor.
- Automatic screw lockout when display frame is disabled.
- Split-flap layer/depth control:
  - Flat
  - Raised / above frame
  - Recessed / behind frame
- Font preset preview in the visual editor.
- Clearer custom font guidance in the editor.
- Clock format help text with examples.
- New Arlanda Express inspired theme slot.

## Changed

- Updated internal card version to `0.1.0-beta.16`.
- Clock defaults to `HH:mm:ss`.
- Text source gets a safe default text when switching back from Clock or Entity.
- Former `Sweden Delight` theme slot now uses an Arlanda Airport / Arlanda Express inspired transit-board look.
- Improved frame, inset and display shell separation.
- Improved upper flap motion timing and depth impression.
- Editor hero text now describes Beta 16 editor/display polish.

## Fixed / Improved

- Reduced confusion around font presets and custom CSS font-family placement.
- Added visual feedback for typography presets.
- Avoided independent screw/frame states that do not make visual sense.
- Added clearer diagnostics for frame and display layer mode.
- Improved recessed/raised display behavior for dashboards where the card background previously filled too much of the visible area.

## Validation focus

Please test:

- Clock source defaults to `HH:mm:ss`.
- Switching Clock → Text restores useful default text.
- Display frame modes render correctly.
- Decorative screws disable when frame is set to `none`.
- Flat / Raised / Recessed layer modes are visibly different.
- Font preset preview updates when preset changes.
- Arlanda Express theme feels more suitable than the old Sweden Delight style.
- Upper flap animation reads more clearly as a flip/fall motion.
- Existing entity and text cards still render correctly.
